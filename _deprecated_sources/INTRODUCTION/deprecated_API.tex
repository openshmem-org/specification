For the \openshmem Specification(s), deprecation is the process of identifying API that is supported but no longer recommended for use by program users. For \openshmem library users, said API \textbf{must} be supported until clearly indicated as otherwise by the Specification. In this chapter we will record the API that has been deprecated, the \openshmem Specification that effected the deprecation, and if the feature is supported in the current version of the specification.
