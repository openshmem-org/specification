The constants that start with SHMEM\_* are for \newtext{both} \Fortran{}
\newtext{and \CorCpp, and they are compile-time constants}. 
\oldtext{For backwards compatibility, the \CorCpp constants may start with
either \_SHMEM\_*  or SHMEM\_*.}\newtext{All constants that start with
\_SHMEM\_* are deprecated and provided for backwards compatibility.
Implementations must include them, and notify the user about deprecation of
their use.}
